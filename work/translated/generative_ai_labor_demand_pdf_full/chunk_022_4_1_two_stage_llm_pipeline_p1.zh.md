# 4.1 两阶段大语言模型流水线

流水线分为两个阶段。在第一阶段，我们使用 Llama-3.1-8b-instant 从每条职位信息中提取任务，并将每个任务与由 Lightcast 从所提取的技能信息构建的技能组进行关联。在第二阶段，我们使用 GPT-5-nano 根据当前可用的生成式 AI 工具能在多大程度上执行或协助每项任务，将提取出的任务归入一个曝光层级。两个阶段所用到的完整提示词见附录 A。  
**第一阶段：从职位描述中提取任务。**  

*职位信息 → 任务 → 曝光标签 索引*  
**第一阶段——任务提取**  
模型：Llama-3.1-8b-instant  
职位信息（标题+描述） → 大语言模型提取 → 每条信息3–10个任务  
任务1（专业技能）权重=2  
任务2（通用技能）权重=1  
任务3（专业技能）权重=2  
关联技能组：Lightcast 技能  

**第二阶段——曝光分类**  
模型：GPT-5-nano  
任务列表（来自第一阶段）+ 职位标题 → 大语言模型分类 →  
* 如果≥50%时间节省？ → E0：无曝光  
* 现成可用的工具？ → E1：直接曝光  
* 否则 → E2：间接曝光  
**聚合——职位信息层面曝光指数**  
α = E1（下限）  
β = E1 + 0.5×E2（主要度量）  
γ = E1 + E2（上限）  
权重：专业技能→2，通用技能→1，每条信息内部归一化

【本段要点】
- 提出一种两阶段大语言模型流水线：第一阶段用Llama-3.1-8b-instant从职位信息中提取任务并匹配技能组；第二阶段用GPT-5-nano对任务进行曝光层级分类。
- 第一阶段每条职位信息提取3–10个任务，并按任务类型（专业技能或通用技能）赋予不同权重（2或1）。
- 第二阶段分类依据：若任务有≥50%时间节省则归为E0（无曝光），否则进一步判断是否有现成通用AI工具（直接曝光E1）或仅有间接潜力（间接曝光E2）。
- 最终在职位信息层面构建曝光指数，包括下限α、主要度量β（E1+0.5×E2）和上限γ（E1+E2），并将权重归一化。
