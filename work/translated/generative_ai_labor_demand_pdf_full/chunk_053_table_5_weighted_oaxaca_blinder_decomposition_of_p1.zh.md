# 表5：总体暴露的加权Oaxaca–Blinder分解：GPT前与GPT后时期

|  | GPT前均值 | GPT后均值 | 可解释部分 | 不可解释部分 |
| :--- | :--- | :--- | :--- | :--- |
| 总体 | 0.404 | 0.389 | −0.010 | −0.005 |
| 初级职位 | 0.435 | 0.419 | −0.011 | −0.006 |
| 中级职位 | 0.388 | 0.374 | −0.008 | −0.005 |
| 高级职位 | 0.595 | 0.571 | −0.018 | −0.006 |

注释：前两列报告了GPT前和GPT后时期的加权平均暴露水平。最后两列报告了前后差异的加权Oaxaca–Blinder分解结果，该分解基于两个时期分别进行的加权回归估计得出。相同的定性模式出现在整个职位等级中。初级、中级和高级职位的平均暴露均有所下降，且每个组中的可解释部分和不可解释部分均为负值。最大降幅出现在高级职位中，暴露水平从0.595降至0.571。高级职位的可解释部分绝对值也最大（−0.018），约占总降幅的75%。对于初级和中级职位，可解释部分占总降幅的比例较小但仍显著。这些结果强化了Kitagawa分解的证据，即组成调整在高级招聘信息中尤为明显，同时表明在控制可观测特征后，所有资历组别的暴露水平均有所下降。

【本段要点】
- 表5展示了GPT前与GPT后时期总体暴露的加权Oaxaca–Blinder分解结果。
- 总体平均暴露从0.404降至0.389，可解释部分为−0.010，不可解释部分为−0.005。
- 所有资历级别（初级、中级、高级）均出现暴露下降，且可解释与不可解释部分均为负。
- 高级职位降幅最大（0.595→0.571），可解释部分（−0.018）占总降幅约75%。
- 初级和中级职位的可解释部分占比相对较小，但仍显著。
- 结论：组成调整在高级招聘信息中尤为明显，且控制特征后暴露下降在所有资历组中均存在。
